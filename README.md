# Automated-Classroom-Sytem
 Automatic Light & Fan Control System

 Overview
An automation system that controls lights and fans based on human presence to reduce electricity wastage.

Features
- Automatic ON/OFF control
- Energy efficient
- Real-time detection

Components Used
- Arduino
- Sensors (IR/PIR)
- Relay Module

 Working
1. Sensor detects human presence
2. Arduino processes input
3. Lights and fan turn ON/OFF automatically
